title: Remark Heading
description: Process headings from your document
---

## Usage

It extracts the headings (table of contents) of a document to `vfile.data.toc`. You may write your
own plugin to convert it into a MDX export.

```ts
import { remarkHeading } from 'fumadocs-core/mdx-plugins';

export default {
  remarkPlugins: [remarkHeading],
};
```

If your heading doesn't have an id, it will add one automatically.

### Custom Ids [#custom-heading-id]

You can customise the heading id with `[#slug]`.

```md
# heading [#slug]
```

### Output

An array of `TOCItemType`.

```json doc-gen:typescript
{
  "file": "./content/docs/headless/props.ts",
  "name": "TOCItemType"
}
```
