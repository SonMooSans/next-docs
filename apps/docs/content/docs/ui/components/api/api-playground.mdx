## Usage

Import it in your MDX documents.

```mdx title="tickets.mdx"
import { API, APIPlayground } from 'fumadocs-ui/components/api';

<API>
  <APIPlayground
    method={'GET'}
    route={'/museum-hours'}
    authorization={{
      name: 'Authorization',
      type: 'string',
      value: 'Basic Og==',
      description: 'The authorization token.',
      isRequired: true,
    }}
  >
    ...
  </APIPlayground>
</API>
```

You also need you update the RootProvider to use your own API base URL.

```tsx
import { RootProvider } from 'fumadocs-ui/provider';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <RootProvider
      api={{
        defaultBaseUrl: 'https://redocly.com/_mock/docs/openapi/museum-api',
      }}
    >
      {children}
    </RootProvider>
  );
}
```
