## List special events

Return a list of upcoming special events at the museum.

### Authorization

<Property name={"Authorization"} type={"Basic <token>"} required={true}>

In: `header`

</Property>

### Query Parameters

<Property name={"startDate"} type={"string"} required={false} deprecated={false}>

The starting date to retrieve future operating hours from. Defaults to today's date.

<span>Example: `"2023-02-23"`</span>

<span>Format: `"date"`</span>

</Property>

<Property name={"endDate"} type={"string"} required={false} deprecated={false}>

The end of a date range to retrieve special events for. Defaults to 7 days after `startDate`.

<span>Example: `"2023-04-18"`</span>

<span>Format: `"date"`</span>

</Property>

<Property name={"page"} type={"integer"} required={false} deprecated={false}>

The page number to retrieve.

<span>Example: `2`</span>

<span>Default: `1`</span>

</Property>

<Property name={"limit"} type={"integer"} required={false} deprecated={false}>

The number of days per page.

<span>Example: `15`</span>

<span>Default: `10`</span>

<span>Maximum: `30`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success |
| `400` | Bad request |
| `404` | Not found |

</APIInfo>

<APIExample>

<Requests items={["cURL","JavaScript"]}>

<Request value={"cURL"}>

```bash
curl -X GET "https://api.fake-museum-example.com/v1/special-events?startDate=2023-02-23&endDate=2023-04-18&page=2&limit=15"
```

</Request>


</Request>
